# E-Commerce Marketplace Project

This repository contains a full-stack multi-vendor ecommerce marketplace project scaffold.

## Backend
- Express.js API server
- MongoDB / Mongoose
- Razorpay / Stripe payment integration
- Notification microservice support
- Swagger API documentation
- Service / controller / repository structure

## Frontend
- React 18 + Vite
- Tailwind CSS
- Responsive checkout and dashboard UI

## Run Backend
1. Install dependencies: `npm install`
2. Create `.env` with MongoDB, JWT, Razorpay, Stripe, and email settings
3. Start server: `npm run dev`

## Logs
- `logs/api.log`
- `logs/error.log`
